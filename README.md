## 🛠️ What is soc-ai-telegram-bot?

soc-ai-telegram-bot is a free program designed to watch your Linux server all the time. It looks for threats and stops them automatically. It uses smart AI tools and connects with Telegram so you get alerts right away.

This tool helps keep your server safe without needing technical skills. It works quietly in the background and lets you stay informed through Telegram messages.

## 🔗 Step 3: Connect Your Linux Server

To monitor your Linux server, you need to link it with the soc-ai-telegram-bot.

1. **Prepare your Linux server:**

   - Make sure it is online and accessible over the network.
   - Confirm you have credentials (username and password or SSH key).
   - Enable log sharing or monitoring services if needed (the app will guide you).

2. **In the Windows app:**

   - Open the soc-ai-telegram-bot program.
   - Click “Add Server” or similar option.
   - Enter your Linux server’s IP address or hostname.
   - Provide your user credentials when asked.
   - Save the connection settings.

The app will test the connection. You should see a confirmation if it links successfully.

---

## 📲 Step 4: Set Up Telegram Alerts

The app uses Telegram to send alerts so you can react quickly.

1. Download and install Telegram on your phone or PC if you don’t have it.
2. Open Telegram and create an account or log in.
3. Search for the Telegram Bot created by this software:
   - In the app, click “Add Bot” or search by the bot’s name (usually shown inside the soc-ai-telegram-bot program).
4. Start the bot by clicking “Start” inside Telegram.
5. Copy the token or key generated by Telegram.
6. Paste this token into the soc-ai-telegram-bot app under the “Telegram Settings” section.
7. Save your settings.

Once this is done, you will receive real-time alerts about your server’s security.
